## what you can do now
- Search and download 
    - YouTube Track
    - YouTube Playlist
    - Spotify Track
    - Spotify Playlist
- Download as 360p, 720p and 160kps
- choose where to store your files
- select some videos from the playlist(don't have to download the whole playlist)
